# API REST de Gestión de Pedidos

API REST para gestionar clientes, productos y pedidos de una tienda.

## Tecnologías usadas

- Java 21
- Spring Boot 3.3.0
- Maven
- PostgreSQL
- Spring Data JPA / Hibernate
- Spring Validation
- Lombok
- JUnit 5
- Mockito
